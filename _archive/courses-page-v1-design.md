## Key Features
1. **Gamification Elements**
   - Season progress tracker
   - Rank progression (Rookie → Starter → Pro → Champ)
   - Visual progress bar

2. **Difficulty Filtering**
   - Three-button filter for beginner/intermediate/advanced

3. **Featured Course Highlight**
   - Large card with thumbnail for featured course
   - Prominent CTA button

4. **Sports-themed Language**
   - "Training Grounds"
   - "Level Up Your Game"
   - "Open Playbook"
   - "Season Progress"
