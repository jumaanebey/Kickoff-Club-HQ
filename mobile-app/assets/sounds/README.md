# Sound Assets

Please add the following sound files to this directory. They should be in MP3 format, 44.1kHz, and under 100KB each.

- `collect_coin.mp3`: Short, satisfying chime for collecting coins.
- `upgrade_start.mp3`: Construction begin sound (hammer/saw).
- `upgrade_complete.mp3`: Fanfare or success sound.
- `drill_plant.mp3`: Whoosh sound for planting drills.
- `drill_harvest.mp3`: Pop + chime for harvesting drills.
- `button_tap.mp3`: Subtle click for UI interactions.
- `error.mp3`: Gentle buzz for invalid actions.
