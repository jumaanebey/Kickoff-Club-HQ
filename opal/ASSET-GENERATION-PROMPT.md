# Opal Asset Generation Prompt for Kickoff Club HQ

Create premium mobile game assets for "Kickoff Club HQ" - a football education and management game app with an arcade-style mobile game ecosystem.

## VISUAL STYLE

- Premium 2.5D isometric perspective (similar to Clash of Clans, Top Eleven, EA Sports mobile games)
- Clean, bold, AAA mobile game quality
- Football/NFL-authentic aesthetics with professional sports atmosphere
- Soft ambient lighting with subtle shadows (15-20% opacity)
- PNG format with transparent backgrounds

## COLOR PALETTE (MUST USE)

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Orange | `#FF6A00` | Main brand/energy color |
| Secondary Forest Green | `#1F6A3E` | Field/success color |
| Navy Blue | `#13274F` | Trust/secondary |
| Gold Accent | `#FFB81C` | Coins/rewards/premium |
| Dark Background | `#0A0A0A` | For contrast reference |

## ASSET CATEGORIES NEEDED

### 1. BUILDINGS (Isometric, 512x512px base)

Create 5-level progression buildings:

- **Training Facility** - Weight room with equipment
- **Medical Center** - Recovery/rehab facility
- **Locker Room** - Team preparation area
- **Scout Tower** - Recruitment center
- **Sponsor Suite** - Business/partnership hub

Each building should show visual upgrades across 5 levels (small → large, simple → detailed)

### 2. UNITS/CHARACTERS (400x500px character cards)

Football player position groups in 3 states (idle, training, ready):

- **Quarterbacks** - Leadership poses, arm raised
- **Running Backs** - Explosive stance, ball tucked
- **Tight Ends** - Blocking/receiving hybrid stance
- **Linebackers** - Defensive intensity, ready to tackle
- **Coaches** - Clipboard, headset, sideline attire

### 3. ICONS (64x64px with transparency)

- Playbook icon (strategy/plays)
- Trophy icon (achievements)
- Helmet icon (equipment)
- Whistle icon (training/drills)
- Football icon (game/match)
- Star icon (rating/premium)
- Clock icon (time/speed-up)
- Shield icon (defense/protection)

### 4. BACKGROUNDS (1290x2796px for mobile)

- **Draft Day** - Stadium with spotlight
- **Championship** - Confetti, celebration
- **Combine** - Workout facility, testing
- **Training Camp** - Outdoor practice

### 5. GAME ELEMENTS

- Power-up items (speed boost, shield, magnet, 2x multiplier)
- Obstacle sprites (tackling dummies, cones, hurdles)
- Collectible items (coins, gems, energy bolts)
- Achievement badges (circular medals with football motifs)

## TECHNICAL REQUIREMENTS

- All assets must have transparent backgrounds
- Clean edges, no artifacts
- Consistent lighting direction (top-left light source)
- High contrast for mobile visibility
- Export at @2x and @3x resolutions when possible

## STYLE REFERENCES

- Supercell games (Clash of Clans, Brawl Stars)
- EA Sports mobile titles
- NFL Clash
- Top Eleven football manager
- Premium casual mobile games with progression systems

## DO NOT INCLUDE

- Generic sports imagery (must be football-specific)
- Realistic photo-style renders (keep it stylized game art)
- Complex backgrounds on individual assets (transparent only)
- Text or logos on assets
